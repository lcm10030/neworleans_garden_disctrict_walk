**Assignment 4**
______________________________________________________________________

For this assignment, I created a walking tour of one of my favorite architectural neighborhoods in New Orleans, LA, the Garden District.
To do this, I created a GeoJSON line string that went by, in my opinion, the most interesting architecture in the neighborhood. Along this line, I placed 5 different stops. 
I also created a GeoJSON polygon representing the Garden District's boundaries and another one for the tour stops.

This map is interactive. By clicking start tour, you will zoom to the tour's start, and clicking next stop will take you to each subsequent stop.

At each stop, a panel on the right side of the screen will display a photo of the location and interesting information about the site's history and architecture. 

Files:
- index.html
- script.js
- styles.css
- gardenboundaries.json - polygon of the Garden District's boundaries
- gardenstops.json - points for the tour stops
- gardenwalk.json - line string of the tour route

Technologies Used:
- Mapbox JL JS
- HTML
- https://geojson.io/
- CSS
